# 漢字をマスターする最速ルート。Japanese Kanji Master。

アプリの使い方
1) 「学習を始める」ボタンを押してください。

2) 漢字を確認した後、下部の「答えを見る」ボタンを押してください。

3) 詳細を確認したら、知っていた漢字は「普通」または「簡単」、難しかった漢字は「難しい」または「復習」で答えてください。

4) 漢字の大まかな形を目で覚えたら、すぐに次の漢字へ進んでください。長時間じっくり学習するのではなく、漫画をめくるようにテンポよく進めましょう。
どうすれば早く覚えられますか？
正確な画数を覚える必要はありません。漢字の大まかな形を目で覚えて、どんどん先へ進みましょう。20個の漢字を10分以内にテンポよく進めるのがおすすめです。

例単語も必ず一緒に覚えてください。

一度に大量に学習するよりも、毎日少しずつ繰り返すほうがはるかに効果的です。
どうしても覚えられない漢字があります
覚えられない漢字は「難しい」で答えて、すぐに次へ進んでください。

「難しい」と答えた漢字は、翌日の復習に再び登場します。

毎日繰り返し見ていれば、ある瞬間に自然と覚えられます。

画数が多くて複雑な漢字は、正確な画数を覚えようとせず、大まかな形をつかんで目に慣らしてください。
自動復習機能
このアプリは学習者の回答を記録し、復習する漢字と復習間隔を自動で設定します。

そのため、別途復習計画を立てる必要はありません。

自動復習機能を信頼して、毎日継続的に学習し、記憶を呼び戻してください。
漢字の難易度
本アプリの漢字難易度は、日本の漢字検定の区分に基づいています。

1〜6段階：小学校の漢字
7〜8段階：中学校の漢字
9〜10段階：高校の漢字
11段階：日常でよく使われる常用外漢字

JLPTを目標とする場合は、以下を参考にしてください。

N5：1段階
N4：2〜3段階
N3：4〜5段階
N2：6〜7段階
N1：8〜9段階
N1高得点：10段階
