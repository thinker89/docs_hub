# FAQ

Saya sudah membeli, tapi tidak bisa mengakses semua kanji
Jika setelah membeli kamu tidak bisa mengakses semua kanji, coba tutup aplikasi sepenuhnya lalu buka kembali.
Pastikan juga koneksi internet kamu stabil.
Jika masalah masih berlanjut, buka [Menu → Pengaturan → Pulihkan Pembelian] dan pulihkan pembelianmu.
Jika kamu menginstal ulang aplikasi atau mengganti perangkat, kamu juga perlu menggunakan [Pulihkan Pembelian].

Pembelian hanya bisa dipulihkan dengan akun Google Play atau App Store yang sama saat melakukan pembelian. Jika kamu menggunakan akun berbeda, masuk ke akun tersebut lalu lakukan pemulihan.
Saya ganti perangkat dan progres saya hilang
Pertama, di perangkat lama, buka [Menu → Pengaturan → Simpan Progres] untuk membuat cadangan.
Simpan file cadangan di tempat yang bisa diakses di perangkat baru (messenger, iCloud, Google Drive, dll.).
Di perangkat baru, buka [Menu → Pengaturan → Pulihkan] lalu muat file tersebut untuk mengembalikan progresmu.
File progres yang disimpan di cloud tidak terlihat
Penyimpanan cloud yang ditampilkan di [Pengaturan → Pulihkan] terkadang tidak dalam kondisi terbaru.
Jika demikian, buka aplikasi cloud yang kamu gunakan (seperti iCloud atau Google Drive) agar perangkat dapat memperbarui dan mengenali file terbaru.
Terlalu mudah—saya ingin melewati level
Jika level saat ini terasa mudah, lebih baik lanjut ke level berikutnya.

1) Ketuk [Semua Kanji] di menu atas
2) Ketuk [menu ⋮] di kanan atas
3) Ketuk [Lewati Level]

Kanji yang dilewati tetap akan muncul sesekali saat review, jadi tidak perlu khawatir. Jika terasa mudah, lanjut saja dengan percaya diri.
Tidak ada suara atau terdengar seperti robot?
Menginstal suara berkualitas tinggi di pengaturan perangkat akan membuat pelafalan terdengar jauh lebih alami.

Tergantung pada perangkat, Anda mungkin perlu menginstal paket suara bahasa Jepang secara manual atau menyesuaikan pengaturan.

[iOS]
Buka [Pengaturan → Aksesibilitas → Konten yang Dibacakan → Suara → Bahasa Jepang → Suara], lalu unduh suara berkualitas tinggi dan pilih sebagai suara yang digunakan.

[Android]
Setelan dapat berbeda tergantung produsen, tetapi biasanya Anda bisa menginstal atau mengganti ke suara berkualitas lebih tinggi melalui menu seperti [Pengaturan → Umum → Bahasa → Teks ke Ucapan] atau yang serupa. Silakan lihat panduan dari produsen perangkat Anda untuk langkah yang lebih rinci.

Jika setelah itu suara masih belum berubah, tutup aplikasi sepenuhnya lalu jalankan kembali.
Tampilkan lebih banyak fitur aplikasi
Aplikasi ini dirancang agar kamu bisa langsung mulai belajar dengan menekan “Mulai Belajar” tanpa pengaturan yang rumit.
Namun, jika kamu ingin lebih banyak opsi, kamu bisa mencoba pengaturan berikut:

1) Mode Gelap
Ubah tampilan ke tema gelap.

2) Mode Prioritas Kata
Aktifkan [Pengaturan → Tampilan Kartu Kanji → Prioritas Kata] untuk menampilkan kata terlebih dahulu di kartu.
Jika kamu adalah pembelajar menengah atau lanjutan yang ingin lebih fokus pada kosakata daripada cerita kanji, cobalah fitur ini.

3) Tampilkan Contoh Secara Default
Jika kamu merasa tidak nyaman harus mengetuk untuk membuka contoh kalimat, aktifkan opsi ini.

4) Atur Jumlah Belajar
Secara default, satu sesi menampilkan 20 kartu kanji, tetapi kamu bisa menyesuaikannya sesuai kebutuhanmu.

5) Simpan Progres Belajar
Progres belajar tidak disimpan di server, jadi pastikan untuk secara berkala mencadangkannya ke cloud pribadimu.
Gunakan fitur ini juga saat berpindah ke perangkat lain.

6) Lihat Kanji Lemah
Buka [Semua Kanji → menu ⋮ → Kanji Lemah Saya] untuk melihat kanji yang sulit kamu ingat.
Kanji lemah ditentukan berdasarkan respons tingkat kesulitanmu.

7) Lewati Level Saat Ini
Jika level saat ini terasa terlalu mudah, kamu bisa melewatinya melalui [Semua Kanji → menu ⋮ → Lewati Level].
Kanji yang dilewati tetap akan muncul sesekali untuk membantu mengingat kembali.
Jadi tidak perlu khawatir—lewati saja level yang mudah dengan percaya diri.
Apakah asal-usul kanji berdasarkan teori akademis?
Beberapa penjelasan mengikuti teori tradisional, sementara yang lain disederhanakan atau dibuat agar lebih mudah dipelajari.
Jika kamu ingin penjelasan yang benar-benar akademis, silakan merujuk ke buku atau sumber penelitian khusus.
Apakah aplikasi ini mengumpulkan data pribadi?
Aplikasi ini tidak mengumpulkan informasi pribadi seperti nama, email, atau kontak.
Namun, data teknis seperti pola penggunaan, progres belajar, dan log error dapat dikumpulkan secara otomatis untuk meningkatkan layanan.

Kamu bisa mencadangkan data belajar ke Google Drive atau iCloud milikmu sendiri jika ingin. Data pribadi yang dikumpulkan oleh layanan tersebut mengikuti kebijakan masing-masing.
Data cadangan mencakup pengaturan aplikasi dan progres belajar, tetapi tidak termasuk informasi pribadi seperti nama atau email.
