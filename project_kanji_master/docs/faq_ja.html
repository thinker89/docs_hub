# FAQ

音声パックがないため、例文の音声を聞くことができません。
端末のメーカー、製造年、または発売国によっては、日本語TTS機能が標準で搭載されていない場合があります。端末の設定の［一般 - 言語 - テキスト読み上げ］またはそれに類似したメニューから、日本語の音声パックを手動でインストールしてください。日本語TTS音声パックの詳しいインストール方法については、各メーカーのマニュアルをご参照ください。
漢字の由来は学説に基づいていますか？
漢字カードに記載されている由来には、伝統的な学説に基づく説明もありますが、学習を効率化するために創作した説明も含まれています。伝統的な学説を正確に学びたい場合は、専門的な論文などもあわせてご参照ください。
学習進捗を保存したいです
アプリを削除したり、別の端末で利用する前に、[メニュー-設定-学習進捗を保存]機能で学習進捗を保存してください。保存先としてiCloudやGoogle Driveを選択すると、より簡単にバックアップと復元ができます。

進捗を復元する端末で[メニュー-設定-復元する]機能を使い、保存したバックアップファイルを読み込んでください。

バックアップを行わずにアプリを削除した場合、アプリ設定および学習進捗は失われ、復元できませんのでご注意ください。
課金したのにすべての漢字を学習できません
決済が正常に完了しているのにすべての漢字を利用できない場合は、まずアプリを完全に終了してから再度起動してください。

問題が続く場合は、[メニュー → 設定 → 購入を復元]をタップして購入履歴を復元してください。

アプリを削除して再インストールした場合や、端末を変更した場合も[購入を復元]を実行してください。

購入履歴はGoogle PlayまたはApp Storeの同一アカウントでのみ復元できます。別のアカウントで決済した場合は、そのアカウントでストアにログインしたうえで購入を復元してください。
個人情報を収集しますか？
本アプリは、氏名、メールアドレス、連絡先など個人を特定できる情報を収集しません。アプリの品質向上のため、利用パターンや学習進捗などの技術的な分析情報が自動的に収集される場合があります。

ユーザーが希望する場合、自身のGoogle DriveまたはiCloudに学習データをバックアップできます。これらのクラウドサービスが収集する個人情報は各プラットフォームのポリシーに従います。バックアップデータにはアプリ設定および学習進捗が含まれ、メールアドレスや氏名などの個人情報は含まれません。
