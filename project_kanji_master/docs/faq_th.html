# FAQ

ฉันซื้อแล้ว แต่ไม่สามารถเข้าถึงคันจิทั้งหมดได้
หากหลังจากซื้อแล้วคุณไม่สามารถเข้าถึงคันจิทั้งหมดได้ ให้ลองปิดแอปให้สมบูรณ์แล้วเปิดใหม่อีกครั้ง
และตรวจสอบว่าการเชื่อมต่ออินเทอร์เน็ตของคุณเสถียรหรือไม่
หากปัญหายังไม่หาย ไปที่ [เมนู → การตั้งค่า → กู้คืนการซื้อ] เพื่อกู้คืนการซื้อของคุณ
หากคุณติดตั้งแอปใหม่หรือเปลี่ยนอุปกรณ์ ก็จำเป็นต้องใช้ [กู้คืนการซื้อ] เช่นกัน

การกู้คืนการซื้อจะทำได้เฉพาะกับบัญชี Google Play หรือ App Store เดิมที่ใช้ซื้อเท่านั้น หากคุณใช้บัญชีอื่น ให้เข้าสู่ระบบบัญชีนั้นแล้วทำการกู้คืน
ฉันเปลี่ยนอุปกรณ์แล้ว และความคืบหน้าหายไป
ก่อนอื่น บนอุปกรณ์เดิม ไปที่ [เมนู → การตั้งค่า → บันทึกความคืบหน้า] เพื่อสร้างไฟล์สำรอง
บันทึกไฟล์ไว้ในที่ที่สามารถเข้าถึงได้จากอุปกรณ์ใหม่ (messenger, iCloud, Google Drive ฯลฯ)
บนอุปกรณ์ใหม่ ไปที่ [เมนู → การตั้งค่า → กู้คืน] แล้วโหลดไฟล์เพื่อกู้คืนความคืบหน้า
ไม่พบไฟล์ความคืบหน้าที่บันทึกไว้ในคลาวด์
พื้นที่จัดเก็บคลาวด์ที่แสดงใน [การตั้งค่า → กู้คืน] อาจไม่ใช่ข้อมูลล่าสุดเสมอไป
ในกรณีนี้ ให้เปิดแอปคลาวด์ที่คุณใช้งานอยู่ (เช่น iCloud หรือ Google Drive) เพื่อให้อุปกรณ์อัปเดตและมองเห็นไฟล์ล่าสุด
มันง่ายเกินไป — ฉันอยากข้ามเลเวล
หากเลเวลปัจจุบันง่ายเกินไป แนะนำให้ไปเลเวลถัดไป

1) แตะ [คันจิทั้งหมด] ที่เมนูด้านบน
2) แตะ [เมนู ⋮] ที่มุมขวาบน
3) เลือก [ข้ามเลเวล]

คันจิที่ข้ามไปจะยังคงปรากฏเป็นครั้งคราวในส่วนทบทวน ดังนั้นไม่ต้องกังวล หากรู้สึกว่าง่าย ก็ไปต่อได้เลย
ไม่มีเสียง หรือเสียงฟังดูเหมือนหุ่นยนต์
ปัญหานี้อาจเกิดขึ้นเพราะอุปกรณ์ของคุณยังไม่ได้ติดตั้งเสียงภาษาญี่ปุ่น หรือกำลังใช้เสียงคุณภาพต่ำอยู่

การติดตั้งเสียงภาษาญี่ปุ่นคุณภาพสูงจะช่วยให้การออกเสียงฟังเป็นธรรมชาติมากขึ้น

[iOS]
1. เปิดการตั้งค่าของอุปกรณ์
2. ไปที่ [การช่วยการเข้าถึง] → [เนื้อหาที่อ่านออกเสียง] → [เสียงพูด]
3. เลือก [ภาษาญี่ปุ่น] แล้วดาวน์โหลดเสียงที่คุณต้องการ
4. ตรวจสอบว่าได้เลือกเสียงภาษาญี่ปุ่นที่ดาวน์โหลดแล้ว

[Android]
1. เปิดการตั้งค่าของอุปกรณ์
2. ใช้แถบค้นหาแล้วพิมพ์ "Text-to-Speech" หรือ "ข้อความเป็นเสียงพูด"
3. เปิดการตั้งค่า "Text-to-Speech" (ข้อความเป็นเสียงพูด)
4. ดาวน์โหลดเสียงภาษาญี่ปุ่น หรือเปลี่ยนเป็นเสียงคุณภาพสูง

หลังจากเปลี่ยนการตั้งค่าแล้ว ให้ปิดแอปให้สนิทและเปิดใหม่อีกครั้ง

หากยังใช้งานไม่ได้ อุปกรณ์ของคุณอาจไม่รองรับ TTS ภาษาญี่ปุ่น
แนะนำฟีเจอร์เพิ่มเติมของแอป
แอปนี้ออกแบบมาให้คุณสามารถกด “เริ่มเรียน” แล้วเรียนได้ทันที โดยไม่ต้องตั้งค่าอะไรซับซ้อน
อย่างไรก็ตาม หากต้องการฟีเจอร์เพิ่มเติม คุณสามารถลองตั้งค่าดังนี้:

1) โหมดมืด
เปลี่ยนหน้าจอเป็นธีมสีเข้ม

2) โหมดเน้นคำศัพท์
เปิด [การตั้งค่า → รูปแบบการ์ดคันจิ → เน้นคำศัพท์] เพื่อให้แสดงคำศัพท์ก่อนบนการ์ด
หากคุณเป็นผู้เรียนระดับกลางหรือสูงที่ต้องการเน้นคำศัพท์มากกว่าสตอรี่ของคันจิ ลองใช้ฟีเจอร์นี้ดู

3) แสดงตัวอย่างอัตโนมัติ
หากการต้องแตะเพื่อเปิดตัวอย่างประโยคไม่สะดวก สามารถเปิดใช้งานตัวเลือกนี้ได้

4) ปรับปริมาณการเรียน
โดยปกติ หนึ่งรอบจะมีการ์ดคันจิ 20 ใบ แต่คุณสามารถปรับให้เหมาะกับตัวเองได้

5) บันทึกความคืบหน้าการเรียน
ความคืบหน้าไม่ได้ถูกเก็บไว้บนเซิร์ฟเวอร์ ดังนั้นควรสำรองข้อมูลไว้ในคลาวด์ของคุณเป็นระยะ
ใช้ฟีเจอร์นี้เมื่อเปลี่ยนอุปกรณ์ด้วย

6) ดูคันจิที่อ่อน
ไปที่ [คันจิทั้งหมด → เมนู ⋮ → คันจิที่อ่อนของฉัน] เพื่อดูคันจิที่คุณจำได้ยาก
ระบบจะจัดหมวดคันจิที่อ่อนตามระดับความยากที่คุณเลือก

7) ข้ามเลเวลปัจจุบัน
หากเลเวลปัจจุบันรู้สึกง่ายเกินไป สามารถข้ามได้ที่ [คันจิทั้งหมด → เมนู ⋮ → ข้ามเลเวล]
คันจิที่ข้ามไปจะยังปรากฏเป็นครั้งคราวเพื่อช่วยทบทวนความจำ
ดังนั้นไม่ต้องกังวล—ข้ามเลเวลที่ง่ายได้อย่างมั่นใจ
ที่มาของคันจิอิงตามทฤษฎีทางวิชาการหรือไม่?
บางคำอธิบายอิงตามทฤษฎีดั้งเดิม ในขณะที่บางส่วนถูกปรับให้เรียบง่าย หรือสร้างขึ้นเพื่อช่วยให้เรียนได้เร็วขึ้น
หากคุณต้องการข้อมูลเชิงวิชาการที่แม่นยำ แนะนำให้ศึกษาจากหนังสือหรือแหล่งข้อมูลเฉพาะทาง
แอปนี้เก็บข้อมูลส่วนตัวหรือไม่?
แอปนี้ไม่เก็บข้อมูลส่วนบุคคล เช่น ชื่อ อีเมล หรือข้อมูลติดต่อ
อย่างไรก็ตาม ข้อมูลทางเทคนิค เช่น รูปแบบการใช้งาน ความคืบหน้าในการเรียน และ log ข้อผิดพลาด อาจถูกเก็บโดยอัตโนมัติเพื่อปรับปรุงบริการ

คุณสามารถสำรองข้อมูลการเรียนไปยัง Google Drive หรือ iCloud ของคุณเองได้ หากต้องการ ข้อมูลส่วนบุคคลที่ถูกเก็บโดยบริการเหล่านั้นจะเป็นไปตามนโยบายของแต่ละบริการ
ข้อมูลสำรองจะรวมการตั้งค่าและความคืบหน้า แต่ไม่รวมข้อมูลส่วนตัว เช่น ชื่อหรืออีเมล
