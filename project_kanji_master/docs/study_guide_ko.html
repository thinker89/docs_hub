# 한자를 마스터하는 가장 빠른 길. 일본어 한자 마스터.
어떻게 학습하면 좋을까요?
1) '학습 시작하기' 버튼을 눌러주세요.
복습 주기를 고려하여 자동으로 선정된 20장의 한자 카드 학습이 시작됩니다.

2) 한자를 확인한 후 하단의 '정답 보기' 버튼을 눌러주세요.

3) 한자의 세부 내용을 확인한 후 하단의 학습 난이도를 선택해 주세요.

이미 알고 있는 한자였다면 '보통'이나 '쉬움'으로 응답해 주세요.
어려운 한자였다면 '어려움'이나 '반복'으로 응답해 주세요.

세 번 이상 '반복' 학습을 해도 어렵다면 '어려움'으로 응답하고 넘어가 주세요.

4) 20장의 카드를 10분 내로 빠르게 훑어보는 방법이 가장 좋습니다. 무리해서 외우지 말고 만화책 보듯 가볍게 읽으며 넘어가 주세요.
도저히 외워지지 않는 한자가 있어요.
세 번 이상 '반복'으로 응답하며 학습했지만 기억에 남지 않는다면 '어려움'으로 응답하고 넘어가 주세요.

'어려움'으로 응답된 한자는 다음날 복습에 다시 등장합니다. 두뇌는 단기간에 무리해서 외운 내용보다, 하루 간격으로 여러 번 반복해서 보여진 내용을 더 오래 기억하려는 성질이 있습니다.

다소 시간이 걸리더라도 매일 반복해서 보는 한자는 어느 순간 저절로 외워집니다.
한자 쓰기도 연습해야 할까요?
아니요. 추천하지 않습니다.

빠르게 읽는 훈련이 훨씬 효과적입니다.

현대 일본어는 글을 쓸 때 대부분 키보드 자동입력으로 해결합니다. 그래서 한자를 읽는 능력이 자연스럽게 쓰기 능력까지 대신해 줍니다.

직업상 손으로 한자를 써야 하거나, 서예 취미가 아니라면 읽는 연습에 집중해 주세요.
한국식 한자 음/뜻도 익혀야 할까요?
네, 익혀두면 도움이 됩니다.

한국식 발음과 일본어 음독을 매칭시켜 익혀두면, 발음과 문맥을 통해 처음 듣는 단어의 뜻을 유추할 수 있는 힘이 키워집니다.
