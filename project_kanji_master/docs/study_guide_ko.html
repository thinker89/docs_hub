# 한자를 마스터하는 가장 빠른 길. 일본어 한자 마스터!

고양이 한자 암기법으로 빠르게 외우세요
매일 보는 길고양이는 '줄무늬가 5개, 꼬리에 하얀 점'처럼 구체적인 형태를 잘 몰라도 보자마자 알 수 있죠?

이제부터 길고양이를 눈에 익히는 것과 마찬가지 방법으로 한자를 공부하기로 해요.

한자의 정확한 모양과 획수를 쓰면서 외우는 방법은 이제 그만두세요.

────────

1) '학습 시작하기' 버튼을 누르세요.

2) 한자를 곁눈질로 빠르게 확인한 후 '정답 보기' 버튼을 누르세요.

3) 한자의 스토리와 그림, 핵심 단어를 살펴보세요.

4) 학습 내용이 어려웠다면 '어려움', 쉬웠다면 '보통' 또는 '쉬움' 버튼을 누르세요.

────────

앱은 사용자의 응답을 기억해요.
'어려움' 한자는 다음 학습에 더 자주 등장해요.
'쉬움' 한자는 학습에 거의 등장하지 않게 돼요.

이 방법으로, 어려운 한자를 매일 자주 마주쳐 익숙하게 만드는 게 바로 고양이 한자 암기법이에요.
도저히 외워지지 않는 한자가 있어요
두세 번 반복해도 외워지지 않는 한자는 '어려움'으로 응답하고 빠르게 넘어가 주세요.

'어려움'으로 응답한 한자는 복습에 다시 등장해요.
매일 반복해서 보다 보면 어느 순간 저절로 외워져요.

복잡한 한자는 정확한 획수를 외우려고 하지 말고 모양을 대강 파악해서 눈에 익혀주세요. 매일 반복이 핵심이에요.
자동 복습
이 앱은 학습자의 응답을 기억하여 다음 학습에서 복습할 한자를 최적의 복습 간격에 따라 제시해요.

따라서 복습 계획을 따로 세울 필요가 없어요.

자동 복습 기능을 믿고 매일 꾸준히 학습을 진행하여 기억을 되살려 주세요.
약점 한자 확인
시험이나 면접 전날, 내 약점 한자를 몰아서 빠르게 복습해야 할 때가 있어요. 아래 방법에 따라 내 약점 한자를 모아서 확인할 수 있어요.

1) 상단 메뉴의 [모든 한자] 터치
2) 모든 한자 화면의 우측 상단 [⋮ 메뉴] 터치
3) [내 약점 한자] 확인

앱은 사용자의 약점 한자를 기억하여 학습할 때 더 자주 보여줘요.
따라서 평소에는 일반 학습만으로도 충분히 복습이 이루어져요.
시험 전날처럼 빠르게 약점만 몰아서 봐야 할 때만 이 메뉴를 이용해 주세요.
한자의 난이도 분류
이 앱의 한자 난이도는 일본 현지 학생들이 한자를 배우는 기준을 따라요.

1~6단계 : 초등학교 한자
7~8단계 : 중학교 한자
9~10단계 : 고등학교 한자
11단계 : 일상에서 많이 쓰는 비공식 한자

JLPT 시험을 목표로 한다면 아래 기준을 참고해 주세요.

N5 : 1단계
N4 : 2~3단계
N3 : 4~5단계
N2 : 6~7단계
N1 : 8~9단계
N1 고득점 : 모든 한자
한자의 원리가 궁금해요
영어를 예로 들면, dislike에서 dis-는 부정, 반대를 뜻하고 like는 좋다는 뜻이니까 합쳐져서 싫다는 뜻이 돼요. preview에서 pre-는 어떤 단어에 붙든 대체로 이전, 미리 등을 뜻하죠.

이와 마찬가지로 한자도 더 작은 단위인 구성자로 쪼개볼 수 있어요. 예를 들어볼게요.

亻: 사람
木: 나무
「休」: 사람(亻)이 나무(木)에 기대어 쉰다. ‘휴식’을 뜻해요.

石: 돌을 뜻해요.
更: 뜻 없이 발음 こう만 나타내요.
「硬」: 돌(石)처럼 ‘단단하다’는 뜻이에요. 更(こう)는 발음만 나타내요.

이처럼 구성자는 뜻이나 발음을 나타내요. 대부분의 한자는 이런 원리로 빠르게 이해할 수 있어요.

또한, 하나의 구성자는 여러 가지 뜻으로 해석될 수 있어요.

口 : ‘입’을 뜻하지만 말, 먹다, 네모난 물건 등을 뜻하기도 해요.
十 : ’10’을 뜻하지만 십자가, 사거리 등을 뜻하기도 해요.

영어에서 dis-, pre-를 따로 외우지는 않는 것처럼, 한자 구성자를 따로 외울 필요는 없어요. 이 앱을 통해 학습을 진행하다 보면 자연스럽게 이해할 수 있어요.
자주 등장하는 구성자가 궁금해요
이 앱을 통해 학습을 진행하다 보면 다양한 구성자를 만나게 돼요.

이제 일본어의 기본기가 튼튼하게 잡힌 상태예요. 계속해서 일본어 달인을 향해 나아가세요.

1) 더 많은 단어 공부하기
이 앱에서 다루지 않은 특수 발음과 고급 어휘를 계속 정복하세요.

2) 문법과 문형 공부하기
단어를 조합하여 긴 문장으로 말하는 훈련을 거듭하세요.

3) 한자 복습하기
한자는 휘발성이 강해서 다시 까먹어요. 이 앱으로 매일 1번, 5분 만에 빠르게 훑어보고 복습을 마무리하세요.

일본어를 하루 1시간 공부한다면 단어 40분, 문법 15분, 한자 5분 정도 시간을 배분하는 걸 추천해요.
