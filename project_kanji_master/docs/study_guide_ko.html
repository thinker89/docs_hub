# 한자를 마스터하는 가장 빠른 길. 일본어 한자 마스터!

고양이 한자 암기법으로 빠르게 외우세요
매일 보는 길고양이는 '줄무늬가 5개, 꼬리에 하얀 점'처럼 구체적인 형태를 잘 몰라도 보자마자 알 수 있죠?

이제부터 길고양이를 눈에 익히는 것과 마찬가지 방법으로 한자를 공부하기로 해요. 한자의 정확한 모양과 획수를 쓰면서 외우는 방법을 버리세요.

────────

1) '학습 시작하기' 버튼을 누르세요.

2) 한자를 곁눈질로 빠르게 확인한 후 '정답 보기' 버튼을 누르세요.

3) 한자의 스토리와 그림, 핵심 단어를 살펴보세요.

4) 학습 내용이 어려웠다면 '어려움', 쉬웠다면 '보통' 또는 '쉬움' 버튼을 누르세요.

────────

앱은 사용자의 응답을 기억해요.
'어려움'으로 표시한 한자는 다음 학습에 더 자주 등장해요.
'쉬움'으로 표시한 한자는 거의 학습에 등장하지 않게 돼요.

이러한 방법으로, 어려운 한자를 매일 자주 마주쳐 익숙해지는 방법이 바로 고양이 한자 암기법이에요.
도저히 외워지지 않는 한자가 있어요
외워지지 않는 한자는 '어려움'으로 응답하고 빠르게 넘어가 주세요.

'어려움'으로 응답된 한자는 다음날 복습에 다시 등장해요.
매일 반복해서 보다 보면 어느 순간 저절로 외워져요.

복잡한 한자는 정확한 획수를 외우려고 하지 말고 모양을 대강 파악해서 눈에 익혀주세요. 매일 반복이 핵심이에요.
자동 복습
이 앱은 학습자의 응답을 기억하여 다음 학습에서 복습할 한자를 최적의 복습 간격에 따라 제시해요.

따라서 복습 계획을 따로 세울 필요가 없어요.

자동 복습 기능을 믿고 매일 꾸준히 학습을 진행하여 기억을 되살려 주세요.
약점 한자 확인
시험이나 면접 전날, 내 약점 한자를 몰아서 빠르게 복습해야 할 때가 있어요. 아래 방법에 따라 내 약점 한자를 모아서 확인할 수 있어요.

1) 상단 메뉴의 [모든 한자] 터치
2) 모든 한자 화면의 우측 상단 [⋮ 메뉴] 터치
3) [내 약점 한자] 확인

앱은 사용자의 응답을 기억하여 약점 한자를 분류해요. 따라서 평소에는 일반적인 학습을 진행하고, 필요한 경우에만 이 메뉴를 이용해 주세요.
한자의 난이도 분류
이 앱의 한자 난이도는 일본 현지 학생들이 한자를 배우는 기준을 따라요.

1~6단계 : 초등학교 한자
7~8단계 : 중학교 한자
9~10단계 : 고등학교 한자
11단계 : 일상에서 많이 쓰는 비공식 한자

JLPT 시험을 목표로 한다면 아래 기준을 참고해 주세요.

N5 : 1단계
N4 : 2~3단계
N3 : 4~5단계
N2 : 6~7단계
N1 : 8~9단계
N1 고득점 : 모든 한자
이 앱의 모든 학습 과정을 마쳤어요
축하합니다! 🎉

이제 일본어의 기본기가 튼튼하게 잡힌 상태예요.
다음 과정을 거쳐 계속 일본어 달인을 향해 나아가세요.

1) 더 많은 단어 공부하기
이 앱에서 다루지 않은 특수 발음과 고급 어휘를 계속 정복하세요.

2) 문법과 문형 공부하기
단어를 조합하여 긴 문장으로 말하는 훈련을 하세요.

3) 한자 복습하기
한자는 휘발성이 강해서 복습을 게을리하면 다시 까먹어요. 이 앱으로 하루 1번, 5분 만에 훑어보고 빠르게 복습을 마무리하세요.

계속해서 일본어를 하루 1시간 공부한다면 단어 40분, 문법 15분, 한자 5분 정도 시간을 배분하는 걸 추천해요.
