# 한자를 마스터하는 가장 빠른 길. 일본어 한자 마스터.

앱 사용 방법
1) '학습 시작하기' 버튼을 눌러주세요.

2) 한자를 확인한 후 하단의 '정답 보기' 버튼을 눌러주세요.

3) 한자의 세부 내용을 확인한 후, 알고 있던 한자는 '보통'이나 '쉬움', 어려운 한자는 '어려움'이나 '반복'으로 응답해 주세요.

4) 한자의 모양을 대강 눈으로 익힌 후 빠르게 다음 한자로 넘어가 주세요. 긴 시간 꼼꼼하게 학습하지 말고 만화책 훑어보듯 빠르게 살펴보고 넘어가며 학습해 주세요.
어떻게 학습하면 빨리 외울까요?
한자의 정확한 획수를 파악할 필요는 없어요. 한자의 대략적인 모양을 눈으로 익히고 빠르게 넘어가 주세요. 20개의 한자를 10분 내로 빠르게 진행해 주세요.

예제 단어들을 반드시 함께 익혀주세요.

몰아서 한번에 대량 학습 하는 것보다 매일 조금씩 반복하는 게 훨씬 효과적이에요.
도저히 외워지지 않는 한자가 있어요
외워지지 않는 한자는 '어려움'으로 응답하고 빠르게 넘어가 주세요.

'어려움'으로 응답된 한자는 다음날 복습에 다시 등장해요.

매일 반복해서 보다 보면 어느 순간 저절로 외워져요.

획수가 많아 복잡한 한자는 정확한 획수를 외우려고 하지 말고 모양을 대강 파악해서 눈에 익혀주세요.
자동 복습 기능
이 앱은 학습자의 응답을 기억하여 복습할 한자와 복습 간격을 자동으로 설정해요.

따라서 복습 계획을 따로 세울 필요가 없어요.

자동 복습 기능을 믿고 매일 꾸준히 학습해서 기억을 되살려 주세요.
한자 쓰기도 연습해야 할까요?
아니요. 추천하지 않아요. 한자의 대략적인 모양을 눈에 익혀 빠르게 읽는 훈련이 훨씬 효과적이에요.

현대 일본어는 글을 쓸 때 대부분 키보드 자동입력으로 해결해요. 그래서 한자를 읽는 능력이 자연스럽게 쓰기 능력까지 대신해요.

직업상 손으로 한자를 써야 하거나, 서예 취미가 아니라면 읽는 연습에 집중해 주세요.
한자의 난이도
이 앱의 한자 난이도는 일본 현지의 한자검정시험 분류에 따라요.

1~6단계 : 초등학교 한자
7~8단계 : 중학교 한자
9~10단계 : 고등학교 한자
11단계 : 일상에서 많이 쓰는 비공식 한자

JLPT 시험을 목표로 한다면 아래 기준을 참고해 주세요.

N4 : 2~3단계
N3 : 4~5단계
N2 : 6~7단계
N1 : 8~9단계
N1 고득점 : 10단계
