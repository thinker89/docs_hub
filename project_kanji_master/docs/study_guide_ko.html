# 한자를 마스터하는 가장 빠른 길. 일본어 한자 마스터.
어떻게 학습하면 좋을까요?
1) '학습 시작하기' 버튼을 눌러주세요.

2) 한자를 확인한 후 하단의 '정답 보기' 버튼을 눌러주세요.

3) 한자의 세부 내용을 확인한 후, 알고 있던 한자는 '보통'이나 '쉬움', 어려운 한자는 '어려움'이나 '반복'으로 응답해 주세요.

4) 20장의 카드를 10분 내로 빠르게 훑어보는 방법이 가장 좋아요.
도저히 외워지지 않는 한자가 있어요
외워지지 않는 한자는 '어려움'으로 응답하고 넘어가 주세요.

'어려움'으로 응답된 한자는 다음날 복습에 다시 등장해요.

매일 반복해서 보다 보면 어느 순간 저절로 외워집니다.
자동 복습 기능
이 앱은 학습자의 응답을 기억하여 복습할 한자와 복습 간격을 자동으로 설정해요.

따라서 복습 계획을 따로 세울 필요가 없어요.

자동 복습 기능을 믿고 매일 1~2회 꾸준히 학습해 주세요.
한자 쓰기도 연습해야 할까요?
아니요. 추천하지 않습니다.

빠르게 읽는 훈련이 훨씬 효과적입니다.

현대 일본어는 글을 쓸 때 대부분 키보드 자동입력으로 해결합니다.

그래서 한자를 읽는 능력이 자연스럽게 쓰기 능력까지 대신해 줍니다.

직업상 손으로 한자를 써야 하거나, 서예 취미가 아니라면 읽는 연습에 집중해 주세요.
한국식 한자 음/뜻도 익혀야 할까요?
네, 익혀두면 도움이 됩니다.

한국식 발음과 일본어 음독을 매칭시켜 익혀두면, 발음과 문맥을 통해 처음 듣는 단어의 뜻을 유추할 수 있는 힘이 키워집니다.
한자의 난이도
이 앱의 한자 난이도는 일본 현지의 한자검정시험 분류에 따라요.

1~6단계 : 초등학교 한자.
7~8단계 : 중학교 한자.
9~10단계 : 고등학교 한자.
11단계 : 일상에서 많이 쓰는 비공식 한자.

JLPT 시험을 목표로 한다면 기준은 다음과 같습니다.

N5 : 1단계.
N4 : 2~3단계.
N3 : 4~5단계.
N2 : 6~7단계.
N1 : 8~9단계.
N1 고득점 : 10단계.
단어와 예문
모든 한자 카드에는 활용 단어가 적혀있어요.

예문 보기를 터치하면 예문도 확인할 수 있어요.

예문을 터치하면 자동음성 발음을 들어볼 수 있어요.

한자가 실제로 어떻게 쓰이는지 단어와 예문을 통해 학습해 주세요.
